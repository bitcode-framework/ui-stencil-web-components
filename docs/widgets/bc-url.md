# bc-url

> URL display with clickable link

## Quick Start

```html
<bc-url></bc-url>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | string | '' | URL |

## Methods

| Method | Returns | Description |
|--------|---------|-------------|
| setValue(value) | Promise<void> | Set URL |
| getValue() | Promise<string> | Get URL |

See [theming](../theming.md).
